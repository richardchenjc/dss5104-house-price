## Reproducing Results

```bash
# 1. Install dependencies
pip install pandas numpy scikit-learn matplotlib seaborn reportlab pypdf

# 2. Run the full analysis pipeline
python analysis.py

# 3. Build the PDF report
python make_report.py
```

`analysis.py` will:
- Load and deduplicate `house_dataset.csv`
- Engineer all features
- Run the Lasso regularisation path analysis
- Train all models (lean Ridge, extended Ridge, RBF/Poly kernels, stacked ensemble)
- Generate all figures to `figures/`
- Save a `results.json` summary

`make_report.py` reads `results.json` and `figures/` to produce `house_price_report.pdf`.

## Key Findings

1. **Dataset duplication**: The raw 9,102-row dataset is an artificial duplication of 4,553 unique sales. This inflated tree-based benchmarks by ~3вЂ“4 pp. All results use the deduplicated dataset.

2. **Feature saturation**: Lasso path analysis shows performance saturates near **20 features**. Going from 20 в†’ 38 features costs less than 0.2 pp MAPE with Ridge regression.

3. **Lean model**: 20 carefully chosen features cover location (smoothed target encoding + interactions), size (log + quadratic), age, renovation, condition, view/waterfront, and rooms.

4. **XGBoost gap**: The default XGBoost configuration is severely overfit (train=5.6%, test=15.5%, gap=+9.9pp). The honest Conservative configuration (gap=+0.57pp) is only 1.9 pp better than our lean Ridge model.
